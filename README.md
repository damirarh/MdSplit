# MdSplit

Split a markdown file into multiple files.

Usage:

```
MdSplit [options]
```

Options:

- `-i|--input <INPUT>` Required. The markdown file to split.
- `-o|--output <OUTPUT>` Required. The output directory.
- `-?|-h|--help` Show help information.

Example

```
MDSplit -i ./single.md -o ./split
```
